## Feature list

### word practice

-   Four input modes: follow-up / recognition / review / dictation
-   Intelligent mode: the memory curve automatically calculates the learned words and deepens the memory through dictation
-   Free mode: no restrictions, plan by yourself

### memorize articles

-   Built-in classic textbooks and books, you can also add and import articles by yourself, providing one-click translation and translation comparison functions
-   Dual mode of follow-up + dictation, input sentence by sentence and automatic pronunciation, making recitation more efficient
-   Supports dictating while listening to strengthen memory

### Collection, error book, mastered

-   Input errors when learning words are automatically added to the wrong word book to facilitate subsequent review.
-   Can be actively added to Mastered and automatically skipped during subsequent learning
-   Can be actively added to collections to consolidate review

### high degree of freedom

-   Rich keyboard sounds
-   Customizable shortcut keys
-   Highly customizable setup options

### Simple and efficient

-   Simple design, modern UI, no ads
-   Refreshing interface and simple operation
-   It is not mandatory to follow any platform

### vocabulary

It has built-in vocabulary libraries for commonly used CET-4, CET-6, GMAT, GRE, IELTS, SAT, TOEFL, postgraduate entrance examination English, professional CET-4 English, and professional CET-8 English.
Try our best to meet the needs of most users for word memorization, and we also welcome the community to contribute more lexicon libraries.

## run

#### Note: This project can be run independently, and the data is saved locally. You need to manually back up the data when changing devices, which does not affect normal use;

This project is based on`Vue`For development, a node environment is required to run.

1.  Install NodeJS, reference[Official documentation](https://nodejs.org/en/download)
2.  The project file is large, it is recommended to use`git clone --depth 1 https://github.com/zyronon/TypeWords.git`The command only clones the most recent commit. Direct download
    The Download ZIP function provided by Github cannot be run
3.  In the project root directory, open the command line and run`npm install`to download dependencies.
4.  implement`npm run dev`To start the project, the default address of the project is[`http://localhost:3000`](http://localhost:3000)
5.  Open in browser[`http://localhost:3000`](http://localhost:3000)to access the project.
6.  implement`npm run build`Package project files
